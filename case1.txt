목적지를 하나의 길로만 도달할 수 없습니다. 누군가에게는 편한 길이 누군가에게는 힘든 길일 수 있습니다. 아이에 맞게 지도 편달하는 것이 가장 중요하다고 생각합니다. 실제 사례입니다.

case 1. 

상황 : 학생이 고난이도 문항들만 풀던 상황 국어 영어 탐구 등 성적은 모두 1등급(9등급제)이었지만, 수학 등급이 3등급(9등급제)으로 부진한 상태 고2 11월이 되어 방치가 어려운 상황

분석 : 아이가 아직 개념에 대한 이해도 없이 높은 등급을 위해 높은 문항만 따라가고 있다는 점을 인식

해결 : 고2 11월에 변화를 주는 것은 어렵겠지만, 현재까지 고수해온 방향이 틀렸음을 인지하고, 개념부터 확실한 공부를 하도록 이야기 함. 기존에 가지고 있던 고난이도 문항을 전부 제외하고, 간단한 문항부터 개념 정리 및 유형 학습을 할 수 있게 도와줌

결과 : 3등급(9등급제) -> 1등급(9등급제)으로 마무리

case 2.

상황 : 학생이 전반을 해서 오게 되었는데, 담당 선생님이 아이가 너무 열심히 한다고 잘 봐달라고 부탁하심. 그러나 아이는 모든 문항을 질문이 없이는 해석이 불가능 하고, 고난이도 문항에 대해 접근 자체가 어려운 상황. 고1 4~5등급 (9등급제)

분석 : 아이는 현재 담당 선생님이 본인의 어려운 모든 것을 해결해줄거라 생각함. 다만 해당 방식을 유지하면 스스로 문제를 푸는 힘이 사라질 수 있음.

해결 : 아이와 지속적으로 소통하며, 식을 쓰는 습관을 하나씩 만들어냄. 초반에는 식을 바로 세워주지 않는 탓에 나에게 질문하지 않는 모습도 보였으나, 1시간에 1문제를 풀어도 좋다는 생각으로 지속적으로 식을 쓰게하고, 왜 그런 식을 썼는 지에 대해, 왜 그 식이 틀렸는 지에 대해 문항 분석을 시킴

결과 : 고2 1학기에 1등급(9등급제)으로 마무리

case 3.

상황 : 아이가 개념에 대해 전혀 부진하고, 중학교 공부를 안한 상태. 수학을 어떻게 해야할지 모르는 상황이며, 중학교 시험에서도 부진한 성적 탓에 고등학교 학습이 어려운 상황

분석 : 기본적으로 수학이 너무 싫은 상황이고, 본인에 대한 자신감이 너무 낮은 상황. 가장 먼저 수학을 해야할 이유가 필요함.

해결 : 아이와 관계 형성 우선. 수학을 원리로 이해해야 하므로, 해당 내용 아이에게 항상 구두로 설명할 수 있도록 지도.

결과 : 고등과정 킬러 문항까지 풀 수 있을 정도로 성장하여 최종 2등급 마무리

case 4.

상황 : 스스로 숙제를 안하는 상황. 큰 목적이 없이 학원을 등하원하는 중. 6등급(9등급제)의 성적 수준이며, 기본적인 곱셈공식도 제대로 못 외우는 상황

분석 : 아이 스스로 할 수 있다고 믿어준다면, 해당 내용을 그대로 다시 가져오고 혼나는 일이 발생

해결 : 집으로 귀가하지 않고, 대다수의 문항을 끝내고 귀가. 과정 중에 아이도 많이 힘들어 하였음.

결과 : 6등급(9등급제)에서 2달만에 수1 2등급(9등급제) 확통 2등급(9등급제) 으로 상승

case 5.

상황 : 아이가 공부를 몹시 잘하는 학생이었음. 국어 영어 수학 등 전과목 1등급에 가까운 점수의 학생. 다만 아이가 풀어지려는 경향이 강함 + 귀찮은 일을 배제하고, 쉬운 방향으로만 접근하려함.

분석 : 아이에게 지속적으로 긴장감이 필요하다고 판단 + 귀찮은 일이 귀찮지 않고 중요한 일임을 인식시켜줄 필요

해결 : 수능까지 한 번의 예외도 없이 모든 일을 규칙으로 만들었으며, 오랜 기간 관계가 형성되더라도, 모든 일을 규칙성에 맞게 접근함. 귀찮은 과정이 왜 중요한 지 직접 식으로 써서 보여주고 적용 시킬 수 있도록 함.

결과 : 안정적으로 1등급(9등급제) 마무리
